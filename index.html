من کی هستم؟

سلام به همه!
من محمد فراز امدادی برنامه نویس ، دیزاینر و توسعه دهنده وب سایت هستم و این هم وب سایت شخصی من هست تا بتوانم خودم را به شما معرفی کنم
زبان هایی که من از آن ها برای ساختن وب سایت هام استفاده می کنم تشکیل شده از:
HTML
CSS
JavaScript
علاوه بر این ها از زبان PHP هم برای نوشتن کد های بک اند وب سایت هایی که می سازم و در اختیار شما قرار می دهم استفاده می کنم
لازم به ذکر است که من از وب سایت Figma برای طراحی وب سایت هایی که درست می کنم استفاده می کنم ، البته طراحی وب سایت هام توسط طناز امدادی صورت می گیرد
فریم ورک هایی که بر روی آن ها مسلط هستم و مورد علاقه من هستند :
tailwindcss برای نوشتن کد های CSS
bootstrap برای نوشتن کد های CSS
React برای نوشتن کد ها JavaScript
و در بیشتر پروژه هایم از آن ها استفاده میکنم.
